# RE-CS-React-Trainee-Profile ⚛️

This is a React Repository 👨‍💻

<br>

## For npm Users 🗒️

Download Dependencies

```
$ npm install
```

Run the app in the development mode

```
$ npm run start
```

<br>

## For Yarn Users 🧶

Download Dependencies

```
$ yarn install
```

Run the app in the development mode

```
$ yarn start
```

The Project would start on your localhost on specified port

<br>
